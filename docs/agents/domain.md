# 领域文档

工程技能在探索代码库时如何消费此仓库的领域文档。

## 探索之前，先读取这些

- **`CONTEXT.md`** 在仓库根目录，或
- **`CONTEXT-MAP.md`** 在仓库根目录（如果存在）——它指向每个上下文一个 `CONTEXT.md`。读取与主题相关的每一个。
- **`docs/adr/`** —— 读取涉及你即将工作的区域的 ADR。在多上下文仓库中，也检查 `src/<context>/docs/adr/` 中的上下文特定决策。

如果这些文件中的任何一个不存在，**静默继续**。不要标记它们的缺失；不要建议提前创建它们。

## 文件结构

单一上下文仓库：

```
/
├── CONTEXT.md
├── docs/adr/
└── src/
```
