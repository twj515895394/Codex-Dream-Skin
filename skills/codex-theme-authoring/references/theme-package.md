# Codex Theme Package

Future standard package format:

```text
my-theme.codex-theme/

├── manifest.json
├── theme.json
├── assets/
│   ├── wallpaper.png
│   └── preview.png
├── install.command
└── README.md
```

The package should be portable between compatible Codex Dream Skin Studio installations.

The package contains theme assets only. It does not contain Codex binaries, credentials, or runtime modifications.
