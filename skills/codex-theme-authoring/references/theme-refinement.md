# Theme Refinement Guidance

## Purpose

Improve an existing Codex theme without rebuilding from zero.

## Common Requests

### Too Bright

Adjust:

- reduce background brightness;
- soften highlights;
- increase text hierarchy;
- reduce large bright areas.

### Too Cold

Adjust:

- move blue tones toward warmer neutrals;
- introduce subtle warm shadows;
- reduce clinical appearance.

### Too Tiring

Adjust:

- lower saturation;
- reduce contrast;
- simplify visual noise;
- improve empty space.

### Too Distracting

Adjust:

- reduce subject prominence;
- simplify background details;
- preserve native Codex content priority.

## Refinement Order

1. Wallpaper crop and composition.
2. Palette and contrast.
3. Theme metadata.
4. CSS customization only if necessary.
5. Renderer extensions only for advanced cases.

Never solve a wallpaper problem by adding unnecessary UI decoration.
