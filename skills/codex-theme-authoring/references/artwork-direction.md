# Theme Artwork Direction

## Goal

Create a Codex background that feels like a workspace environment, not a screenshot.

## Composition rules

Default layout:

- Left side: calm empty area for Codex heading and controls.
- Center: visual transition area.
- Right side: main subject or visual focus.

Preferred aspect ratio:

- 16:9
- 2560x1440 minimum
- 3200px wide preferred

## Avoid

Never include:

- fake Codex windows;
- buttons;
- text labels;
- code editors;
- chat bubbles;
- application chrome.

## Long-session design principles

Prefer:

- soft contrast;
- balanced shadows;
- low saturation;
- comfortable reading environment;
- natural depth.

Avoid:

- aggressive neon;
- extreme contrast;
- busy textures;
- distracting focal points.
