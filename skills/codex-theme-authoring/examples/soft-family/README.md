# Soft Family Theme Example

## Concept

A warm and low-fatigue Codex workspace theme inspired by family photography.

## Design Direction

- Soft Scandinavian minimalism
- Warm companionship
- Calm creative workspace
- Long-session friendly colors

## Visual Rules

- Keep the main subject on the right side.
- Preserve left-side negative space for Codex text overlays.
- Avoid embedding UI elements into the wallpaper.

## Palette Direction

- Mist blue
- Warm white
- Soft gray
- Muted accent colors
